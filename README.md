# Kroix Jones Portfolio

Static personal portfolio for Kroix Jones, built with HTML, CSS, JavaScript, and a Three.js hero scene.

## Run Locally

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Deploy

This is a static site deployed from the repository root with Cloudflare Pages.

Production domains:

- `https://kroixjones.dev`
- `https://www.kroixjones.dev` redirects to `https://kroixjones.dev`
